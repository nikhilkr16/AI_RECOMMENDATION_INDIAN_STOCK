Frequently Asked Questions

Is IndianStockAI completely free to use?

Yes. The dashboard runs on Streamlit Community Cloud (free). The AI uses Groq's free tier (14,400 requests/day with llama3-70b) and Google Gemini's free tier as backup. No credit card required for any component.

Which stocks does IndianStockAI cover?

All NSE and BSE listed stocks are supported via yFinance. The heatmap and screener cover all Nifty 50 constituents. Enter any NSE symbol (e.g., TCS, RELIANCE, INFY) to analyse it.

How accurate are the AI trading signals?

The signals are generated from technical indicators (RSI, MACD, Bollinger Bands, etc.) and AI reasoning. They are for educational purposes only and should not be the sole basis for any investment decision. Always consult a SEBI-registered financial advisor.

What makes this different from other stock apps?

Most free stock apps do not have an AI summary feature, a WhatsApp-shareable digest, or a sector heatmap. IndianStockAI bundles all of these together using zero-cost APIs — no paywalls.

How do I run this locally or contribute?

The source code is open on GitHub. Clone the repo, install requirements with pip install -r requirements.txt, add your free Groq API key, and run streamlit run AdvanceStockAnalysis.py.
